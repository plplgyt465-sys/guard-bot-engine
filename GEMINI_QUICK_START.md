# دليل البدء السريع - Gemini

## 🎯 الهدف
استخدام Gemini كمزود AI الوحيد والحصري

---

## ⚡ البدء في 3 دقائق

### الخطوة 1️⃣ - احصل على مفتاح API

👉 اذهب إلى: **https://aistudio.google.com/apikey**

```
1. اضغط على "Create API Key"
2. اختر مشروع أو أنشئ واحد جديد
3. سيظهر المفتاح - انسخه!
```

---

### الخطوة 2️⃣ - أضفه في التطبيق

```
1. افتح تطبيق Guard Bot Engine
2. اضغط على أيقونة ⚙️ (الإعدادات) في الزاوية العلوية
3. اختر تبويب: 🤖 مزود الذكاء
4. الصق المفتاح في حقل Gemini
5. اضغط على "فحص الكل" للتحقق
6. اضغط على "حفظ الإعدادات" ✅
```

---

### الخطوة 3️⃣ - ابدأ الاستخدام

```
🎉 Gemini جاهز الآن!
اكتب أي رسالة - Gemini سيرد تلقائياً
```

---

## 🆘 المشاكل الشائعة

### ❌ "Invalid API Key"
```
✓ تأكد من نسخ المفتاح بالكامل بدون مسافات
✓ المفتاح الجديد قد يحتاج دقائق للتفعيل
```

### ❌ "No Response"
```
✓ تحقق من اتصال الإنترنت
✓ جرب مفتاح API آخر
✓ انتظر دقيقة وحاول مجدداً
```

### ❌ "Quota Exceeded"
```
✓ تحقق من الرصيد في: console.cloud.google.com
✓ قد تحتاج لترقية الحساب
```
