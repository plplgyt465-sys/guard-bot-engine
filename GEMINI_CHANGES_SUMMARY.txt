4. QUICK START
================================================================================

1. GET API KEY:
   → Go to: https://aistudio.google.com/apikey
   → Click "Create API Key"
   → Copy the key

2. CONFIGURE IN APP:
   → Open app
   → Click "إعدادات" (Settings)
   → Go to "🤖 مزود الذكاء" (AI Provider) tab
   → Paste API key
   → Click "حفظ الإعدادات" (Save Settings)

3. START USING:
   → Gemini is ready immediately!
   → All AI requests use Gemini exclusively
